# Bem-vindo ao seu laboratório de Engenharia de Dados

Você é analista VOIP em transição de carreira. Este sistema foi montado para acelerar sua entrada na área de dados através de prática real, não teoria.

## Como funciona

Cada LAB é um exercício que você executa na sua máquina. Não existe "assistir" aqui — você digita, roda, erra, corrige e aprende.

Siga a ordem abaixo. Não pule etapas.

## Ordem dos Labs

## Regras de uso

1. **Tente antes de ver o gabarito.** Cada LAB tem gabarito no final, separado por uma linha. Só olhe depois de tentar.
2. **Errou? Ótimo.** Leia a mensagem de erro, pense no que causou, tente de novo.
3. **Atualize o PROGRESSO.md** quando concluir cada lab.
4. **Me acione como professor** quando travar: cole o erro e o que você tentou. Não cole só o erro.
