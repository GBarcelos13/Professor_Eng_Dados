# 📚 Professor Eng. Dados

Material de estudo de **Engenharia de Dados** com laboratórios práticos em SQL, Python e pipelines ETL — organizado como um curso guiado por um professor virtual.

## Sobre o Projeto

Este repositório funciona como um ambiente de aprendizado estruturado para Engenharia de Dados. O conteúdo foi organizado em módulos progressivos, do SQL básico até pipelines ETL com Spark e NoSQL.

## Conteúdo

| Lab | Tema | Tecnologias |
|-----|------|-------------|
| LAB 01 | SQL Básico | PostgreSQL |
| LAB 02 | SQL Avançado | PostgreSQL, Window Functions |
| LAB 03 | SQL com Northwind | PostgreSQL, Schema real |
| LAB 04 | Python Fundamentos | Python |
| LAB 05 | Python + Banco de Dados | Python, psycopg2, SQLAlchemy |
| LAB 06 | Pipeline ETL | Python, Pandas |
| LAB 07 | NoSQL com MongoDB | MongoDB, PyMongo |
| LAB 08 | Apache Spark | PySpark |
| LAB 09 | Projeto Final | Stack completa |

## Como Usar

```bash
# Clone o repositório
git clone https://github.com/GBarcelos13/Professor_Eng_Dados.git
cd Professor_Eng_Dados

# Comece pelo guia de setup
cat PROFESSOR_VIRTUAL/01_SETUP.md
```
